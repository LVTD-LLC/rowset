What does not change when you upgrade.

Pro raises the hosted capacity. It does not move you to a different product path or change the private-by-default model.

Agent handoff first

Copy the setup prompt, provide a bearer key, and let a trusted agent create or discover datasets.

Private authenticated access

REST and MCP stay behind your account boundary. Public previews are separate and read-only.

Stable row operations

Agents can inspect schemas, create rows, update rows, export snapshots, and keep index values explicit.

Human review surface

The dashboard remains for setup, current state, exports, settings, and public preview review.
