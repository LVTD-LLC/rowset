Agents do not need another fixed app. They need somewhere to keep state.

Most tools start by deciding the object for you: ticket, lead, roadmap item, article, issue. Rowset starts with the durable part underneath all of them: indexed rows that an agent can create, inspect, update, export, and explain.

The agent invents the model

It can create a dataset for whatever job is in front of it, with headers and an index column that match the workflow instead of a vendor template.

Rowset keeps the rows stable

Authenticated MCP and REST tools give agents a predictable way to read, mutate, paginate, export, and share structured data.

Humans keep control

The browser stays a control surface for account setup, recent state, exports, public previews, and recovery when an agent needs help.

Use it as many products without turning Rowset into many products.

The product surface is intentionally small. Agents bring the domain logic; Rowset gives them the structured backend.

The loop is small on purpose.

Rowset does not try to automate your whole company from the browser. It gives agents the missing persistence layer between conversations, tools, files, and humans.

  1. define The agent creates a dataset with the headers and index column the job requires.
  2. operate MCP or REST tools let it list, read, create, update, and delete rows inside the authenticated account boundary.
  3. review Humans can inspect datasets, export CSV or Parquet, and enable optional read-only public previews.
  4. adapt When the workflow changes, the agent can create a better structure instead of fighting a fixed app model.

Built for trusted agents, not public scraping.

Bearer auth

Hosted MCP and REST use API-key bearer authentication, usually through a private environment variable or client secret store.

Ownership boundaries

Agents operate only on datasets available to the authenticated profile. Public previews are separate sharing settings.

Machine-readable truth

Dataset keys, headers, semantic column types, row operations, exports, and preview URLs stay explicit.

Human escape hatch

When an agent needs review, the dashboard and public previews give people a readable view without taking over row operations.
